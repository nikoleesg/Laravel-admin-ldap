laravel-admin extension
======

# Laravel-admin LDAP



````
LDAP_CACHE=false
LDAP_LOGGING=true
LDAP_CONNECTION=default
LDAP_HOST=<domain-controller>
LDAP_USERNAME="cn=<username>,ou=Managed Users,dc=<domain>,dc=com,dc=<country>"
LDAP_PASSWORD="<password>"
LDAP_PORT=389
LDAP_BASE_DN="dc=<domain>,dc=com"
LDAP_TIMEOUT=5
LDAP_SSL=false
LDAP_TLS=false
LDAP_SASL=false
````
